Homepage

Get Methods

Get by Title

URL: /getTitle

Query Params:

title: title of the book

Returns:

A single book with a matching title

Get by Genre

URL: /getGenre

Query Params:

genre: genre to search for

Returns:

array of book objects that contain the genre

Get Random Book

URL: /getRandom

Query Params:

none

Returns:

A single random book from the dataset

Post Methods

Rate Book

URL: /rateBook

Query Params:

title: title of book to rate

review: rating of the book

Returns:

the updated book

Add Book

URL: /addBook

Query Params:

title: title of book to add. required

author: author of the book to add

country: country of origin of book to add

link: link to wikipedia page of book to add

pages: number of pages of book to add

year: release year of book to add

genre: genre of book to add

Returns:

the updated book
